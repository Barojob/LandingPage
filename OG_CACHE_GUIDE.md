OG 이미지 캐시 초기화 방법:

1. 페이스북 디버거: https://developers.facebook.com/tools/debug/
   - URL 입력 후 'Debug' 클릭
   - 'Scrape Again' 버튼 클릭

2. 트위터 카드 밸리데이터: https://cards-dev.twitter.com/validator
   - URL 입력 후 검증

3. 링크드인 포스트 인스펙터: https://www.linkedin.com/post-inspector/

4. 카카오톡 공유 디버거: 카카오 디벨로퍼스에서 확인

5. 일반적인 캐시 초기화:
   - URL 끝에 ?v=1 같은 쿼리 파라미터 추가
   - 예: https://www.inteuk.com/ogImage.png?v=1
